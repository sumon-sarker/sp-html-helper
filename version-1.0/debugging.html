Developer Console Debugging

Default configuration

SP.debug();


Example 1 [Show Logs to Developer Console]

var SP = new SpHtmlHelper({
    debugMode   : true, /*Enable debug mode, Otherwise Debug mode is disabled*/
    screenSize  : 2000 /*Define your SP device screen, [In pixel]*/
});



//All of your code goes here



SP.debug(); /*Call this function to the last line of your script*/

Result


Example 2 [Show AddMenu Logs to Developer Console]

var SP = new SpHtmlHelper({
    debugMode   : true, /*Enable debug mode, Otherwise Debug mode is disabled*/
    screenSize  : 2000 /*Define your SP device screen, [In pixel]*/
});

SP.AddMenu({
    menuLogo        : 'dist/img/sp-html-helper.png',
    targetContainer : 'body',
    menuItems       : [
        {
            title       : 'SpHtmlHelper1',
            link        : '#1',
            css_class   : ''
        },
        {
            title       : 'SpHtmlHelper2',
            link        : '#2',
            css_class   : ''
        },
    ]
});

SP.debug(); /*Call this function to the last line of your script*/

Result
