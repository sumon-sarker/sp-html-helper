Menu Configuration

Default configuration

SP.MenuConfig({
	multiSiteMenu	: false,
	languageString 	: '/en/',
	copyTargetMenu 	: {
		copy 		: false,
		targetTag 	: ""
	}
});

Multilingual Menu



Example 1 [Adding japanese language menu]

SP.MenuConfig({
	multiSiteMenu	: true,
	languageString 	: '/jp/'
});

Copy existing website menu as SP Menu Items



Example 2 [Copy website menu]

SP.MenuConfig({
	copyTargetMenu 	: {
		copy 		: true,
		targetTag 	: "body ul"
	}
});


Full Configuration for above 2 example dist/js/sp-html-helper-config.js

SP.MenuConfig({
	multiSiteMenu	: true,
	languageString 	: '/jp/',
	copyTargetMenu 	: {
		copy 		: true,
		targetTag 	: "body ul"
	}
});

Sp.AddMenu({
    targetContainer : 'body' /*Menu will add to body*/
});
